# Ultrasonic Servo Mapper

## Description
This project uses an Arduino, an HC-SR04 ultrasonic sensor, and a servo motor. The system measures the distance of a nearby object and automatically rotates the servo motor based on that distance. 

Using the `map()` function, an object's distance (from 0 to 100 cm) is smoothly translated into the servo's rotation angle (from 0 to 180 degrees). The closer the object, the more the servo turns!
