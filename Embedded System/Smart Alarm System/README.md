# Smart Alarm System

## Description
This project is a straightforward security alarm system built with an Arduino. It utilizes a PIR (Passive Infrared) sensor to monitor the surrounding area. Whenever movement is detected, the system immediately triggers a visual alert (lighting up a red LED) and an audible alarm (sounding a Piezo buzzer at 1000Hz). When the area is clear, the alarm resets and turns off automatically.
